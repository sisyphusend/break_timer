# BreakTimer - 休息提醒应用

一个跨平台的桌面应用程序，使用的是QT框架，用于提醒您定时休息，保护视力。

## ✨ 功能

- **后台运行**: 应用程序在系统托盘中运行，不干扰您的工作。
- **灵活的定时器设置**:
  - **间隔模式**: 按固定的时间间隔（例如，每30分钟）提醒。
  - **CRON 模式**: 支持标准的 CRON 表达式，实现高度灵活的定时计划。
- **全屏提醒**:
  - 到达预定时间后，显示一个半透明的全屏覆盖层。
  - 覆盖层中央显示一个图标，提醒您休息。
  - 点击覆盖层或按键即可轻松关闭。
- **跨平台**: 使用 C++/Qt 构建，支持 Windows 和 Linux (Wayland/X11)。
